# Producción · checklist universal

## Pre-producción

- [ ] Casting CR confirmado (2-3 actores por video) · agreement de imagen firmado
- [ ] Locaciones scout + permisos
- [ ] Props list + compra (cuaderno, sobre oficial, uniforme, etc.)
- [ ] Wardrobe review — nada muy glossy, todo "real CR"
- [ ] Guion impreso para actor · 30 min rehearsal
- [ ] Logos EffiCommerce SVG + PNG 3x
- [ ] UI mockups EffiCommerce aprobados por design lead

## Producción

- [ ] Cámara DSLR/mirrorless 4K (sony/canon) · lente 35mm + 85mm
- [ ] Iluminación natural + rebote; NO estudio glossy
- [ ] Audio lavalier o shotgun
- [ ] Grabar múltiples takes por shot
- [ ] B-roll ambiental de locación

## Post-producción

- [ ] Edición en Premiere/DaVinci · color grading warm LATAM
- [ ] Captions burned-in español (font Inter, bold, contraste alto)
- [ ] Music license confirmado (Epidemic/Artlist comprobante)
- [ ] Logo overlay esquina superior izquierda durante todo el video
- [ ] Disclaimer "Dramatización" si aplica (actor)
- [ ] Thumbnail/cover optimizado
- [ ] Export 3 formatos: MP4 H.264 1080x1920 (Meta/TikTok), 1080x1350 (4:5), 1080x1080 (1:1)
- [ ] Export SRT de subtítulos separado

## QA

- [ ] Duración dentro del target (+/- 2s)
- [ ] Volumen audio normalizado a -12 LUFS
- [ ] Captions legibles en mobile (test en celular real)
- [ ] CTA final claro y visible
- [ ] Compliance Ley 7472: sin "garantizado", "#1", "único"
- [ ] Compliance Meta Ads: sin claims prohibidos, sin urgencia falsa
- [ ] Peso ≤ 30MB por archivo
